# Cosa caricare su GitHub

Apri questa cartella e carica su GitHub il suo contenuto, non la cartella intera.

La repo deve mostrare subito:

```text
index.html
logo.svg
app.js
styles.css
firebase-config.js
firebase.json
vercel.json
README.md
server.py
functions/
```

Se vedi `carica-su-github-pulito/index.html` dentro GitHub, hai caricato la cartella sbagliata: devi entrare nella cartella e selezionare tutti i file.

Su Vercel lascia `Root Directory` vuota.
